# Tsumiki キャラクター ビジュアル仕様書

本番イラスト発注・差し替え時の基準書。MVP では `src/characters/*.tsx` の SVG がこの仕様の実装です。
アプリ内のキャラ切り替えは `src/characters/index.ts` の `DEFAULT_CHARACTER_ID`(初期値)と、設定画面から可能。

共通の感情差分は 7 種:

| キー | 状況 | 共通の演出方針 |
|---|---|---|
| `cheer` | 朝・今日まだ未完了 | 明るく後押し。目はぱっちり、口は開いた笑顔 |
| `proud` | 今日完了/継続順調 | 誇らしい。胸を張る、目を閉じたにっこり、キラキラ |
| `worried` | 昼過ぎ・未完了 | そわそわ・お願い。眉が八の字、口は小さく波線、汗ひとつ |
| `tearful` | 夜・未完了(途切れ寸前) | 涙目。目に大粒のハイライト+涙、口はへの字 |
| `sad` | ストリークが途切れた | しょんぼり。目を閉じて下向き、彩度を落とす |
| `relieved` | 途切れから復帰 | 泣いて喜ぶ。涙+満面の笑み |
| `celebrate` | レベルアップ/バッジ | 大喜び。腕(相当パーツ)を上げ、紙吹雪・星 |

## 案A マメ(Mame)— 芽吹きの妖精

- **コンセプト**: 「継続の豆」から芽を出した新芽の妖精。毎日のタスク=水やり。ストリーク=育った日数。
- **性格**: 健気で前向き、少し甘えん坊。あなたの水やりが生きがい。頑張った日は全力で誇らしがる。
- **背景ストーリー**: キャリアという大樹の種。5 つの枝(技術・ヒアリング・推進・調整交渉・アウトプット)を伸ばして大樹になる夢を持つ。水やり(タスク)を続ければ育ち、1日忘れると葉がしおれてしまう。
- **口調**: やわらかい。「〜だよ」「いっしょにやろ!」。夜は「今日が終わっちゃうよぅ…」
- **体型**: そら豆型の丸いボディ(縦:横 ≒ 1.1:1)。身長比でボディ 80%、頭頂の双葉 20%。手足は短い丸ナブ。
- **カラー**:
  - ボディ: クリーム `#F6EDD6` / おなか: `#FCF7E8`
  - 双葉・手足: 若葉グリーン `#7BC96F`(影 `#5BA854`)
  - 頬: ピンク `#F5B8B8` / 輪郭線: ダークブラウン `#4A3B32`(太め 3〜4px @200px)
- **輪郭**: 全パーツ丸みのみ。鋭角は使わない。
- **表情差分の特徴**:
  - `cheer`: 目=黒丸+白ハイライト、口=開いた半円笑顔、双葉ピン!
  - `proud`: 目=閉じたアーチ(∩∩)、口=にっこり、周囲にキラキラ2つ
  - `worried`: 眉=八の字、目=小さめ、口=波線、頭に汗1滴、双葉やや垂れ
  - `tearful`: 目=大粒うるうる(ハイライト2つ)+涙、口=への字、双葉しおれ気味
  - `sad`: 目=閉じて下向きの弧、双葉が完全にしおれて垂れる、体色をややくすませる
  - `relieved`: 涙が流れつつ満面の笑み、双葉が半分回復
  - `celebrate`: 両手を上げ、口を大きく開けた笑顔、周囲に紙吹雪・星、双葉から花が一輪
- **成長メタファー(拡張案)**: ストリーク 0-6 日=双葉 / 7-29 日=本葉 / 30-99 日=つぼみ / 100 日〜=花

## 案B ホムラ(Homura)— 継続の炎の精

- **コンセプト**: ストリークの 🔥 と直結した「やる気の種火」の精。燃料はあなたの行動だけ。サボると小さくなり、消えそうになる。
- **性格**: 熱血応援団長。普段は威勢がいいが、燃料が切れかけると急に弱気になるギャップ。
- **背景ストーリー**: あなたが最初のタスクを完了した瞬間に灯った炎。長く燃えるほど大きく、青白く気高くなる。消えても「種火」は残っていて、もう一度行動すれば再点火できる。
- **口調**: 「いくぞ!」「燃えてきたぜ!」。夜は「……このままだと、消えちゃうよ」
- **体型**: しずく型の炎(縦:横 ≒ 1.3:1)。外炎+内炎の 2 層。底は波形。腕は小さな炎の突起。
- **カラー**:
  - 外炎: `#FF7A3C` → 先端 `#FF5A2C` / 内炎コア: `#FFD23F` → `#FFF3B0`
  - 目: 白 `#FFFFFF`(炎の中で光る)/ 口: `#B33B18` / 火の粉: `#FFB25E`
  - 弱った時: 全体を `#FFA98C`/`#FFE9A8` 側に淡く、サイズ 80%
- **輪郭**: 輪郭線なし(発光体)。レイヤーの色差で形を出す。
- **表情差分の特徴**:
  - `cheer`: 目=まっすぐな白楕円、口=ニカッ、火の粉 2 つ
  - `proud`: 炎 110%、目=閉じてドヤ顔、頂点がピンと立つ、火の粉 4 つ
  - `worried`: 炎 90%、先端がくにゃっと曲がる、眉八の字、汗(蒸気)1 つ
  - `tearful`: 炎 75%、うるうる目+涙(涙で「じゅっ」と蒸気)、先端がしおれて垂れる
  - `sad`: 炎 60%・彩度ダウン(灰がかったオレンジ)、目を閉じ、煙がひとすじ
  - `relieved`: 炎 100% に戻り、涙+笑顔、再点火の火花
  - `celebrate`: 炎 120%、両腕(炎の突起)を上げ、火の粉と星が舞う
- **成長メタファー(拡張案)**: ストリークが伸びるほど 炎の色がオレンジ → 金 → 青白 に変化

## 案C ツミ(Tsumi)— つみきの守り人(第3案・アプリ名との統一案)

- **コンセプト**: 「継続はつみき」。毎日のタスク=ブロックを 1 個積むこと。ストリーク=塔の高さ。頭の上に積まれた小さな塔がそのまま継続日数のメタファーになる。
- **性格**: 律儀でけなげな職人気質。積むのが生きがい。「くずれちゃう」ことを何より恐れるが、崩れても「また積めばいい」と立ち直りは早い(復帰導線がポジティブ)。
- **背景ストーリー**: 昔の宮大工が「続ける心」を宿して彫った木の子精霊。持ち主の毎日の行動を 1 個ずつブロックにして積み上げ、キャリアの塔を建てている。5 色のブロック=5 つのスキル領域。
- **口調**: 「〜です!」「今日のいちだん、積みましょう!」。夜は「今日のぶんが……積めずに終わっちゃいます…」
- **体型**: 角丸立方体の木の子(縦:横 ≒ 1:1)。頭の上に小さなブロック 2〜3 個(ストリーク表現)。短い手足。
- **カラー**:
  - ボディ: 木肌 `#E8C99B`(影 `#C9A26E`)/ 顔まわり: `#F7E8CF`
  - 頭上ブロック: 5 領域色(`#5B8DEF` `#9C6ADE` `#F2994A` `#56CCF2` `#6FCF97`)からローテーション
  - 頬: `#F0A98F` / 輪郭線: 焦げ茶 `#59422F`
- **輪郭**: 角丸の四角ベース。木目ライン 1〜2 本。
- **表情差分の特徴**:
  - `cheer`: ぱっちり目+笑顔、頭上ブロックがまっすぐ
  - `proud`: 閉じ目にっこり、頭上ブロック +1 個キラーン
  - `worried`: 眉八の字、頭上ブロックが斜めに傾ぐ(崩れそう!)
  - `tearful`: うるうる+涙、ブロックがさらに傾いてグラグラ線
  - `sad`: 閉じ目でうつむき、ブロックが足元に崩れ落ちている
  - `relieved`: 涙+笑顔、崩れたブロックを 1 個拾って頭に載せ直している
  - `celebrate`: 両手を上げ、ブロックが虹色に発光+紙吹雪
- **差別化**: 案A=成長、案B=情熱に対し、案Cは「積み上げ・喪失の可視化」がもっとも直接的。アプリ名 Tsumiki と世界観が一致する。

## アプリアイコン

各案とも `assets/character/icon-<id>.svg`(1024×1024)に、ブランド背景+キャラ正面顔アップの構成で用意。
`npm run icon -- <mame|homura|tsumi>` で `assets/icon.png` ほかに書き出し(sharp 使用)。

- マメ: 背景 若葉グラデ `#A8E063→#7BC96F`、中央にマメの顔+双葉
- ホムラ: 背景 夜色 `#2B2140`、中央に燃えるホムラ(発光)
- ツミ: 背景 生成り `#FFF3DC`、中央にツミ+頭上ブロック

## 本番イラスト差し替え手順

1. 上記仕様で各感情 7 枚(正方形・背景透過 PNG 1024px 推奨)を用意
2. `assets/character/<id>/<emotion>.png` に配置
3. `src/characters/<id>.tsx` の `Svg` コンポーネントを `Image` 描画に差し替え(1 ファイルの変更で完結)
4. 通知添付画像は初回起動時に自動で再生成される(`IMAGE_VERSION` を上げる)
